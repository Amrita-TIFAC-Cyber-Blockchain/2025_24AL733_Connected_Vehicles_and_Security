It is best practice to run this in a clean virtual environment so the dependencies do not conflict with your other projects.

   ```bash
   python -m venv venv
   ```
2. Activate the virtual environment:
   ```bash
   venv\Scripts\activate
   ```

### Step 3: Install the Dependencies
Based on Jules's summary, the simulation relies on three specific external libraries. Install them using pip:

```bash
pip install networkx pulp simpy streamlit plotly pandas
```
* **`networkx`**: Builds the directed graph topology.
* **`pulp`**: Solves the Integer Linear Programming (ILP) math to calculate the TAS offsets.
* **`simpy`**: Runs the discrete-event simulation for packet routing and gate timings.
